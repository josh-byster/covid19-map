# Interactive COVID-19 Outbreak Map 
[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

## Installation

```
git clone git@github.com:josh-byster/covid19-map.git
npm i
```

## Usage

### Development server

```bash
npm start
```

### Production build

```bash
npm run build
```
